巴哈論壇排行嵌入展示站

這個站提供即時預覽、可直接複製的 iframe 語法，以及三種適合不同欄位的榜單版型。 嵌入頁面本身會讀取最新 JSON，不需要你的網站再額外處理資料。

Live Preview

版型範例

每個預覽都是真正的嵌入頁 iframe。下方的語法可直接貼到部落格、CMS 或任意 HTML 區塊。

Sidebar

側欄榜單

適合 280 到 360px 邊欄。密度高、資訊清楚，適合首頁與文章頁右側欄。

查看大圖

Banner

橫幅榜單

接近巴哈熱門榜單的橫向卡片感，適合首頁主欄、專題頁或寬版區塊。

查看大圖

Inline

文章內嵌榜單

高度較低，適合插入文章內文中段或一般內容欄，不會壓縮閱讀節奏。

查看大圖

Parameters

嵌入參數

第一版官方只提供 iframe，可用查詢字串切換版型與分類。

layout

sidebar、banner、inline

category

mobile、pc、lifestyle

limit

顯示筆數，建議 3 到 12 筆之間

theme

目前支援 light，其餘值會回退到預設
